
Place the following JS files here for FULL offline functionality (exact names recommended):

- chart.min.js            (Chart.js v4.x)
- xlsx.full.min.js        (SheetJS 0.18.5)
- pdf.min.js              (pdf.js 3.10.111)
- tesseract.min.js        (Tesseract.js v5)
- papaparse.min.js        (PapaParse 5.4.1)

These are open-source libraries. Download their minified builds and drop them into this folder,
then rebuild the APK. The app loads from local assets and requires no internet.
